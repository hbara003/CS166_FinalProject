# CS166 Final Project
## How to Run
1. download on Linux
2. run 
  > . ./postgresql/startPostgreSQL.sh
  to start the DB
3. run 
  > . ./postgresql/createPostgreDB.sh
  to create the DB
4. > . ./java/compile.sh
  to compile the main code
5. > . ./java/run.sh $LOGNAME_DB 9998 $LOGNAME
  to run the code
